# Drawjs
React component to help you add drawing to your react project

#### Features
 - Mouse/ track pad drawing
 - Pen options
    - Thickness
    - Color
 - Export as json or image


 ### Examples 
 ```jsx 
 import { DrawCanvas } from "@luke-e-gibson/drawjs"
 
 export function App() {
  return (
    <div>
      <DrawCanvas width={800} height={800}>
    </div>
  )
 }
 ````
